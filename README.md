# eBPF/XDP Stealth Port-Scan Killer 🛡️

A high-performance Intrusion Prevention System (IPS) built with eBPF and XDP, designed to run directly inside the Linux Kernel (Ring 0). 

## 📌 Overview
This proof-of-concept was developed as a 2nd-year Computer Science system engineering project to explore low-level network security. It intercepts and analyzes network packets at the network driver level—before they even reach the Linux networking stack or `iptables`.

Currently, the IPS is configured to detect and instantly drop **TCP SYN Scans** (stealth scans, often generated by tools like `nmap`), mitigating reconnaissance phases of an attack with zero latency.

## ⚙️ Architecture
* **The Muscle (C Code):** Injected into the kernel via XDP. It parses Ethernet, IP, and TCP headers in real-time, tracking SYN flags without ACK flags. Malicious packets are destroyed at the NIC level (`XDP_DROP`).
* **The Brain (Python/BCC):** Runs in user-space, compiles the C code using LLVM/Clang, loads it into the kernel, and reads shared memory (eBPF Maps) to display real-time attack telemetry.
